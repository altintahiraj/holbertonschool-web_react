# holbertonschool-web_react

A collection of React projects and exercises covering the core concepts of modern React development.

# Overview

This repository contains different modules focused on learning and practicing React fundamentals. Each folder represents a specific topic, starting from the basics and gradually moving to more advanced concepts.

The goal is to build a solid understanding of how React works and how to structure real applications.

Topics Covered
React introduction
Components and structure
Props and data passing
State management
React Hooks
Immutable data handling
Project Structure

# Each folder represents a separate topic:

/react_intro – basic React concepts and setup
/react_component – creating and using components
/react_props – passing data between components
/react_state – managing state in components
/react_hooks – using modern React hooks
/React_Immutable – working with immutable data
Getting Started

# To run any project:

cd <folder-name>
npm install
npm start

Make sure you have Node.js installed on your machine.

Purpose

This repository is part of the Holberton School curriculum and is focused on hands-on learning. Each module is designed to reinforce key concepts through practical examples.

# Notes
Each folder can run independently
